JS에는 불리언(Boolean)이라는 자료형(데이터타입)이 있다.
불리언은 true 또는 false를 값으로 가진다.
예를들어

1 + 1 > 1 // true
1 + 1 < 1 // false

위와같이 비교연산자 <, >, <=, >=, ==, ===, !=, !== 를 사용하여 좌변과 우변을 비교하여 true, false를 평가한다.

x > y  : x가 y보다 크다.
x < y : x가 y보다 작다.
x >= y : x가 y보다 크거나 같다.
x <= y : x가 y보다 작거나 같다.

x == y : 동등 비교 ; x와 y의 값이 같음.
x === y : 일치 비교 ; x와 y의 값과 타입이 같음.
x != y : 부동등 비교 ; x와 y의 값이 다름.
x !== y : 불일치 비교 ; x와 y의 값과 타입이 다름.
