API를 이용하여 기상청에서 제공하는 날씨 정보를 가져오고 그 정보를 이용해 현재 날씨를 알려주는 웹애플리케이션을 제작한다.
API를 사용하는 과정에서 직접 코드를 작성하여 if ... else 문, var 변수, ajax 등의 문법을 학습하고 API, JSON등 기초 프로그래밍 개념을 알아볼 것이다.

index.html 파일과 weather.js파일이 있으며 weather.js 파일에서 코드를 작성하면 된다.
index.html 파일을 먼저 보고 weather.js 파일을 보면서 코드를 따라치면 된다.
주석 (//, /**/, <---->)에 지금 하고있는 프로젝트의 설명과 문법설명, 각 코드가 어떤 의미이고 어떤 역할을 하는지 적혀있다.
주석에서 'XXX.txt'를 참고하라고 하면 '참고' 폴더에 있는 txt파일을 참고하라.

콘솔창을 여는 방법은 브라우저에서 'F12'를 누르면 나오는 개발자 도구(DevTools)에서 console 탭을 찾아 클릭하면 된다.
